# Online Bookstore / Library System (Java Project)

This is a simple console-based Java project that simulates an **online library/bookstore system**. It allows adding books, registering users, searching books, borrowing and returning books, and viewing available and borrowed books.

The project is created for learning and demonstrating core **Object-Oriented Programming (OOP)** concepts in Java.
